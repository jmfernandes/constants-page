Planck Constant

How To Use

Reference the url "http://www.constants.datanab.net/physics/planck_constant_json" in order to extract the JSON data. For example, if you wanted to get the variable in a Python script you would do:
import JSON
import urllib
aquire = urllib.urlopen("http://www.constants.datanab.net/physics/planck_constant_json") unpacked = aquire.read()
data = json.loads(unpacked)
Now, you are free to reference data['value'] to use the number.
